## 프로젝트 개요
### 배경 Background
인도 전자상거래 기업의 멀티채널 판매 데이터에서 발견된 주요 문제
- 148,012건의 거래 중 고수익 상품이 명확하지 않아 자원 분산
- 배송 완료율이 채널별로 88.8% vs 0%로 극심한 격차
- 일부 상품은 60일 이상 미판매되어 재고 낭비
- 전사적 수익성 최적화 전략 부재

### 목표 Goal
1. 7,113개 SKU 중 고수익 세그먼트 식별로 자원 배분 명확화
2. 배송 채널별 성과 비교 분석으로 시스템 문제 규명
3. 카테고리별 수익성 격차 해소 방안 도출
4. 실행 가능한 액션 플랜 제시

### 비즈니스 영향

- 고수익 상품 관리 강화 시 연 2-5% 매출 증가 (2백만~4백만 달러)
- 배송 시스템 개선으로 고객 만족도 10-20% 향상
- 휴면 상품 정리로 연 10-15% 재고 관리 비용 절감
- 예상 총 임팩트: 3백만~5백만 달러 추가 이익
