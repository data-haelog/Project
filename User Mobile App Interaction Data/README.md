# 모바일 앱 사용자 행동 분석 프로젝트일 앱 사용자 행동 분석 프로젝트

### 개발개요
사용자 이탈률 증가 문제 해결을 위한 데이터 기반 리텐션 분석 프로젝트로, 60,471명의 사용자 인터랙션 데이터를 분석하여 주요 이탈 지점을 식별하고 개선 전략을 도출. D1 리텐션 3.0%의 심각한 온보딩 문제를 발견하고, 퍼널 분석을 통해 첫 단계에서 85% 이탈이 발생하는 것을 확인하여 구체적인 UX 개선 방안 제시.

### 구현기술
- **언어**: Python
- **데이터 분석**: pandas, numpy, scipy
- **시각화**: matplotlib, seaborn, plotly
- **통계 분석**: scipy.stats (t-test, 통계적 유의성 검정)
- **분석 기법**: 코호트 분석, 퍼널 분석, 리텐션 분석, 세그멘테이션

***

#### 1. 데이터 전처리 및 EDA
- 100,000건의 사용자 인터랙션 데이터 전처리 (결측치 처리, 이상치 제거)
- 23개 컬럼 데이터 정제 및 파생변수 생성 (days_since_first, user_type 등)
- 시간대별, 요일별, 디바이스별 사용자 행동 패턴 탐색적 분석

#### 2. 리텐션 및 코호트 분석
- D1/D7/D30 리텐션 지표 계산 및 업계 평균 대비 성과 평가
- 주별 가입 코호트 그룹별 리텐션 추이 분석 (4주차 평균 7.8%)
- 사용자 세그멘테이션 (Light/Medium/Heavy User) 및 행동 특성 분석

#### 3. 퍼널 분석 및 이탈 지점 식별
- 사용자 행동 단계별 전환율 분석 (최종 전환율 14.5%)
- 주요 이탈 지점 식별: 첫 클릭 단계에서 85% 사용자 이탈 발견
- 단계별 병목 구간 시각화 및 개선 포인트 도출

#### 4. 시각화 및 인사이트 도출 
- 10개 이상의 시각화 차트 제작 (리텐션 그래프, 코호트 히트맵, 퍼널 차트 등)
- 비즈니스 액션 아이템 5가지 도출 및 예상 임팩트 정량화
- 경영진 대상 분석 결과 리포트 작성

***

- **분석 기법**:
  1. **리텐션 분석**: D1/D7/D30 지표 계산
  2. **코호트 분석**: 주별 가입 그룹별 리텐션 추적
  3. **퍼널 분석**: 단계별 전환율 및 이탈률
  4. **세그멘테이션**: Quantile 기반 사용자 분류
